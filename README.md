PackWISE ist keine einfache Checklisten-App. Das Ziel ist es, **nicht jedes Mal bei Null anzufangen**
### Das Grundprinzip 
1.  **Ausrüstung (Bestand)**: Zuerst legst du deine Ausrüstungsgegenstände in der App an (Bestand). 
2.  **Vorlagen (Templates)**: Aus diesem Bestand baust du Vorlagen. Eine Vorlage definiert eine Standard-Ausrüstung
3.  für einen bestimmten Einsatzzweck (z. B. "Wandern Sommer Hütte").
4.  **Touren (Trips)**:       Wenn du verreist, legst du eine Tour an. PackWISE sucht die passende Vorlage und
5.  erstellt eine Kopie für diese spezifische Reise.
6.  **Packliste**:            Die konkrete Liste für deine Tour. Änderungen hier betreffen nur diese eine Tour.
7.  **Workflow**:             Die App begleitet dich durch die Phasen
8.    **Vorbereitung** (Packen),
9.    **Durchführung** (Unterwegs sein) und
10.   **Nachbereitung** (Lernen).
